# 此刻·此地 — 用户使用指南

> 用相机在真实世界留下信，让别人走到那个地方才能看见。

---

## 一、这是什么

**此刻·此地**是一款基于真实地理位置的信件互动应用。你在某个地点拍下照片、写一封信投递出去，后来的人只有走到那个地方、举起相机对准你当时的角度，信才会从取景器中浮现。

它像一个跨越时空的"寻宝游戏"——你发现别人留下的情感痕迹，也留下自己的。

---

## 二、初次使用

### 2.1 设置身份

打开应用后，会进入入门引导。你需要：

1. **选择头像**：从 20 个 emoji 头像中选一个
2. **设置昵称**：这个名字会出现在你写的每一封信上（最多 12 个字）
3. 点击「进入」完成设置

> 昵称和头像之后可以在首页点击头像随时修改。

### 2.2 首页看板

设置完成后进入首页，你会看到：

| 区域 | 说明 |
|------|------|
| **顶部问候** | 根据时间显示"早安/下午好/夜深了" |
| **统计卡片** | 我寄出的信 / 收到的回响 / 附近信件 三组数字 |
| **成就徽章** | 根据写信数量解锁的成就 |
| **活跃热力图** | 近 30 天的写信活跃情况 |
| **探索按钮** | 核心入口，点击进入地图 |
| **我的信件列表** | 最近 5 封信，点击可阅读全文 |
| **最新回响** | 别人对你信件的回复 |
| **FAB 浮动菜单** | 右下角的 ✉️ 按钮，快速写信 / 信匣 / 偶遇 |

## 三、三种信件

### 3.1 📮 公开信 — 众人的"到此一游"

**适合场景**：旅游景点、咖啡馆、街头转角——你想让路过的人看到你的感想。

**发信**：
1. 打开相机拍照
2. 写标题和正文（可选 AI 润色）
3. 选择类型「公开」
4. 可选：开启「场景对齐」——收信人需要对准你拍摄的角度才能看到

**收信**：
1. 在地图上看到附近的信封标记
2. 走近目标地点（20 米内）
3. 打开相机取景器
4. AR 信封浮现 → 对齐角度 → 阅读正文

---

### 3.2 ⏳ 时光胶囊 — 写给未来的自己

**适合场景**：毕业前、跨年夜、某个值得纪念的地方——给一段时间后的自己留句话。

**埋下**：
1. 拍照 → 写信
2. 选择类型「给自己」
3. 设置解锁时间（选择日期或默认 24 小时后）
4. 可选：邀请好友「共埋」——输入好友昵称（逗号分隔）
5. 可选：开启「对齐解锁」——必须回到原地才能打开

**打开**：
- 到达解锁时间后，会收到通知
- 回到原地（如开启对齐）→ 对准角度 → 解锁
- 多人共埋时，全部人打开后才能看到彼此的留言

---

### 3.3 🔒 密信 — 两个人的秘密

**适合场景**：给特定的人留一封信，只有 Ta 输入正确口令才能看到。

**发信**：
1. 拍照（这作为"线索照片"，帮助收信人寻找）
2. 写信
3. 选择类型「密信」
4. 输入接收人昵称
5. 系统自动生成 8 位口令

**发出口令**：
- 信件创建后，会自动弹出分享界面
- 点击「复制口令」发送给收信人
- 也可以通过二维码分享

**收信**：
1. 收到口令后，在地图页面搜索框旁输入口令
2. 获取线索照片 + 大致范围
3. 根据线索找到那个地方
4. 进入范围后直接看到信

## 四、地图探索

### 4.1 查看附近信件

点击首页的「探索附近地图」按钮，进入地图视图：

- **地图标记**：每个信封图标代表一封信
  - 金色 = 公开信
  - 沙漏 = 时光胶囊
  - 锁定 = 密信
  - 带数字的 = 同一地点多封信折叠
- **底部列表**：按距离排序显示附近所有信件
- **筛选**：点击"公开/给自己/密信"切换类型

### 4.2 走近一封信

1. 点击列表中的信封 → 地图聚焦 → 显示路线
2. 走近到 10 米内 → 底部出现「📷 打开相机查看」
3. 点击进入 AR 取景模式

### 4.3 搜索地点

顶部搜索框输入地点名称（依赖高德地图API）。

### 4.4 模拟定位（桌面调试用）

在地图页面点击 🎯 按钮，然后点击地图任意位置模拟 GPS 定位。再次点击 🎯 退出模拟。

## 五、AR 相机取景

这是本应用最核心的体验。

### 5.1 启动相机

从地图进入、或点击 FAB 菜单的「写一封信」都会打开相机。

### 5.2 AR 信封浮现

当你在目标信件 20 米范围内时：

| 距离 | 效果 |
|------|------|
| >20m | 不显示，提示继续走近 |
| 10-20m | 小光点方向指示 |
| <10m | 信封浮现，随对齐度变清晰 |

信封在画面中像"浮在空中"一样——这是通过手机陀螺仪追踪你的朝向实现的，类似 Pokémon GO 的 AR 模式。

### 5.3 场景对齐

如果发信人开启了「场景对齐」，你需要对准原始拍摄角度：

1. 取景器底部显示对齐进度条
2. 调整手机角度，让画面接近原始照片
3. 进度条上升至 90%+ → 信封完全解锁

信封有四个阶段：
- **0-30%**：虚线轮廓，金色光晕闪烁
- **30-60%**：轮廓清晰，发信人头像浮现
- **60-90%**：信纸徐徐展开，文字逐渐可辨
- **90-100%**：完全清晰，照片浮现，可以打开

### 5.4 如何写信

1. 在相机页面点击「📷 拍照写信」
2. 填写标题和正文
3. 选择情绪标签（温柔/俏皮/深情/怀念/期待/感谢）
4. 可选：点击「✨ AI润色」让 AI 帮你润色文字
5. 选择信件类型（公开/给自己/密信）
6. 配置类型相关选项
7. 点击「投递」

> **提示**：如果在没有目标信件的区域打开相机，底部显示「📝 在此留下第一封信」——任何人都能成为第一个在某地留下信的人。

## 六、阅读信件

### 6.1 打开一封信

通过 AR 解锁 / 地图列表 / 信匣 点击信件进入阅读页面：

- **照片**：发信人拍摄的现场照片
- **正文**：信的内容
- **发信人信息**：昵称 + 头像
- **位置**：信件投递地点
- **情绪标签**：发信人选择的心情

### 6.2 留下回响

在阅读页面底部输入框写下回复，对方会在首页收到回响通知。

### 6.3 删除自己的信

自己的信右上角显示 🗑️ 删除按钮。信匣中也可以长按自己的信来删除。

## 七、信匣（收藏管理）

首页 FAB 菜单 → 📬 我的信匣：

- **四个标签页**：
  - 📮 我的信 — 你寄出的所有信
  - 🗺️ 发现的 — 你发现过的别人的信
  - ⏳ 胶囊 — 所有时光胶囊
  - 🔒 密信 — 所有密信
- **排序**：点击右上角按钮切换「最新/最早/情绪」
- **长按删除**：长按自己的信可删除

## 八、偶遇（随机发现）

首页 FAB 菜单 → 🎲 偶遇一封：

- 随机展示附近的一封信
- 可以「换一封」或「打开看看」
- 适合"随便逛逛"的场景

## 十、常见问题

### Q: 为什么相机画面是黑的？

A: 摄像头需要 HTTPS 安全环境或 localhost。如果在手机上用 HTTP 访问，需要配置 HTTPS 证书。桌面浏览器通常直接支持。

### Q: AR 信封不出现？

A: 可能原因：
1. 当前位置没有附近信件——去地图页查看
2. 手机不支持陀螺仪——AR 效果会降级为固定位置显示
3. 浏览器未获得摄像头权限——检查设置

### Q: GPS 定位不准怎么办？

A: 民用 GPS 精度通常 5-20 米。如果偏差较大：
1. 确保在室外开阔环境
2. 等待几秒让 GPS 稳定
3. 系统设置中允许应用获取精确位置

### Q: AI 润色不工作？

A: AI 润色需要 DeepSeek API 密钥。如果未配置，应用会自动使用本地模板库作为降级方案。

### Q: 数据会丢失吗？

A: 信件默认存储在浏览器 IndexedDB 中。如果后端同步服务在运行，数据会自动同步到服务器。清除浏览器数据会导致本地信件丢失。

### Q: 怎么删除账号/退出？

A: 首页点头像 → 编辑资料 → 修改昵称和头像。目前不支持完全删除账号数据，可在浏览器设置中清除站点数据。

## 十一、小贴士

- 🔋 AR 模式较耗电，建议在需要时使用
- 📸 写信时拍的照片建议拍摄有明显特征的角度，方便后来者对齐
- 🎯 桌面端开发时可以使用地图上的模拟定位功能测试
- 📡 近距离（<10m）效果最佳，可以走到信的位置再打开相机
- 💡 成为某个地点的"第一封信"是一个有趣的成就
